# Your Adventure Starts Here

Travel booking website built with HTML, CSS, and JavaScript.

## Overview

This project displays popular destinations, allows users to add trips to a shopping cart, calculate totals (including tax and booking fee), and submit travel reviews.

## Version History

- Version 1.0 (First release): April 10, 2025
- Version 2.0 (Updated release): March 30, 2026

## Main Features

- Destination catalog rendered dynamically with JavaScript.
- Search, price filter, and sorting controls.
- Currency selector (CAD, USD, EUR) with live conversion.
- Shopping cart with add/remove actions and automatic total calculation.
- Review form with star rating and review list.
- Local storage persistence for cart, reviews, and selected currency.
- Toast notifications for user feedback.
- Responsive layout for desktop, tablet, and mobile.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES5-style constructors and DOM APIs)

## Project Structure

- `index.html`: main page structure
- `styles/style.css`: all styles and responsive rules
- `js/script.js`: app logic (catalog, cart, reviews, currency, storage)
- `images/`: project images

## How To Run Locally

1. Clone or download this repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser (or use Live Server extension).
